# -ros2-line_tracer
Jetson 보드의 퍼블리셔노드에서 영상을 캡쳐하여 토픽전송하고

Jetson보드의 섭스크라이버 노드에서 양쪽 바퀴의 속도명령을 계산하여 다이내믹셀로 전송
